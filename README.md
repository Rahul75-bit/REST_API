# User Management System

A Role-Based User Management System built using Spring Boot, Spring Security, JWT Authentication, and React.

## Features

### Authentication
- JWT Based Authentication
- Secure Login
- BCrypt Password Encryption

### Role Based Access Control

#### Admin
- Create Users
- View All Users
- Delete Users
- Assign Roles

#### Manager
- Create Tasks
- Assign Tasks To Users
- View Assigned Tasks

#### User
- Login
- View Profile
- View Assigned Tasks

## Technologies Used

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 Database
- JWT Authentication
- Maven

### Frontend
- React
- Vite
- Fetch API
- CSS3
